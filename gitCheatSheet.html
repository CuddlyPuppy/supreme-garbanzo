Some easy commands for git

  1. git init initiate git repository
  2. git add filename will add file to index for commit
  3. git commit -m "add new file" commits the file added to the index
  4. git status checks if file was added to repository
  5. git remote add origins html://www.blahblahblah.com add remote repository address
  6. git push origin master push changes
  7. git branch shows the branches
  8. git checkout -b sub1 makes and moves to branch
  9. git push origin branchname
